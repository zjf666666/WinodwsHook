# Windows Sercuity Guard

### 一、简介

​	这个项目的初衷是学习windows hook技术，顺便了解一些安全相关知识。但写着写着希望将这些学习的内容集成成一个软件，于是便决定搭一个框架来整合这些内容。

### 二、业务需求

1. windows进程、文件、注册表、网络监控
2. API调试
3. 行为分析
